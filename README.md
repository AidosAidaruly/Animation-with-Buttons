# Animations with Buttons 🎮

A Unity 3D project built for a Game Development course.

## Features
- 🏆 Win animation with celebratory yellow lighting
- 💀 Fail animation with dark red lighting
- 💥 Explosion particle effect
- 🎮 Interactive UI buttons to trigger animations

## Built With
- Unity 6 (URP)
- C#
- Mixamo animations
- Unity Particle System

## How to Run
1. Clone the repository
2. Open project in Unity 6
3. Open SampleScene
4. Press Play and use the buttons!
